# url-shortener

API to make short urls.

## Live demo

[Url Shortener](https://sleepy-brushlands-54298.herokuapp.com/)

## How to run

```(sh)
npm run start            #run the app
npm run start:dev        #run the app in dev mode
```

## Tech stack

- Node / Express
- MongoDB / Monk
- Swagger / Swagger UI
- Heroku
